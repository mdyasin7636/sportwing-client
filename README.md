# SportWing

## Here are 5 Best Features of My Project:

- Admin, Instructor, Student Dashboard.

- Instructor can add class.

- Student can Booked class and pay.

- Data is stored in MongoDB.

- Stripe payment method Used to pay for classes.

## Technologies
- React Router Dom
- Tailwind CSS
- Daisy UI
- Node.JS
- Express.JS
- MongoDB
- Stripe
